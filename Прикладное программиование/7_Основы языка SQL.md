
1. Разница между CHAR(n), NCHAR(n), VARCHAR(n) и NVARCHAR(n)

Ответ:

**CHAR(n)** - строка фиксированной длины (n символов), дополняется пробелами  
**NCHAR(n)** - то же, но для Unicode-символов (поддержка разных языков)  
**VARCHAR(n)** - строка переменной длины (до n символов), экономит место  
**NVARCHAR(n)** - VARCHAR для Unicode-символов

2. Отличия реляционных отношений от таблиц в SQL

Ответ:

**Реляционные отношения** (теоретические):

- Неупорядоченное множество кортежей
    
- Нет дубликатов строк
    
- Атрибуты неупорядоченные
    

**Таблицы SQL** (практические):

- Строки могут быть упорядочены
    
- Могут содержать дубликаты
    
- Столбцы имеют порядок

12. Разница между JOIN операторами

Ответ:

**INNER JOIN** - только совпадающие строки из обеих таблиц  
**LEFT JOIN** - все строки из левой таблицы + совпадающие из правой  
**RIGHT JOIN** - все строки из правой таблицы + совпадающие из левой  
**FULL JOIN** - все строки из обеих таблиц  
**CROSS JOIN** - декартово произведение (все комбинации)

7. Отличия коррелированного и некоррелированного подзапроса

Ответ:

**Некоррелированный** - выполняется один раз, не зависит от внешнего запроса  
**Коррелированный** - выполняется для каждой строки внешнего запроса, ссылается на его столбцы
