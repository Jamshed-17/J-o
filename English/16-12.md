**1. Arithmetic operator — A mathematical function that is used to perform a calculation, eg add, subtract, multiply and divide.**  
*Арифметический оператор — это математическая функция, которая используется для выполнения вычислений, например, сложение, вычитание, умножение и деление.*  
**TRUE**

*Логический оператор — это оператор, предназначенный для работы с логическими значениями. Четырьмя наиболее распространенными логическими операторами в программировании являются логическое И, логическое включение и логическое отрицание.*  
**FALSE**

**3. Global variable — A variable that is only accessible within a specific part of a program.**  
*Глобальная переменная — это переменная, доступная только в определенной части программы.*  
**FALSE**  
*Правильный ответ (англ.):* A global variable is accessible from anywhere in the program.  
*Правильный ответ (рус.):* Глобальная переменная доступна из любого места программы.

**4. High-level programming languages — This is a computer programming language used to write programs and they need to be translated into machine code through a compiler, interpreter or assembler.**  
*Языки программирования высокого уровня — это языки программирования, используемые для написания программ, которые необходимо переводить в машинный код с помощью компилятора, интерпретатора или ассемблера.*  
**TRUE**

**5. Local variable — A variable that is accessible from anywhere in a program.**  
*Локальная переменная — это переменная, доступная из любого места программы.*  
**FALSE**

*Правильный ответ (рус.):* Локальная переменная доступна только внутри функции или блока, где она объявлена.
