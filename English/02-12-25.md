
1. **Algorithm — to run a computer program.**
   - **False**  
   - Исправлено: Algorithm — a set of step-by-step instructions to solve a problem or perform a task.  
   - **Перевод:** Алгоритм — набор пошаговых инструкций для решения задачи или выполнения задания.  


2. **Programming — the process of writing computer software.**
   - **True**  
   - **Перевод:** Программирование — процесс написания компьютерного программного обеспечения.  


3. **Pseudocode — it usually expresses instructions by using the application’s or tool’s rules and syntax, combined with simple control structures such as loops and if/then expressions**
   - **False**  
   - Исправлено: Pseudocode — it usually expresses instructions in a human-readable, informal way, without strict syntax rules, using simple control structures such as loops and if/then expressions.  
   - **Перевод:** Псевдокод — обычно выражает инструкции в удобочитаемой, неформальной манере, без строгих правил синтаксиса, с использованием простых управляющих структур, таких как циклы и выражения if/then.  


4. **Machine language — a language resembling an imperative programming language and used as a notation to describe algorithms and/or program structure**
   - **False**  
   - Исправлено: Machine language — a low-level programming language consisting of binary digits (0s and 1s) that a computer’s CPU executes directly.  
   - **Перевод:** Машинный язык — низкоуровневый язык программирования, состоящий из двоичных цифр (0 и 1), которые непосредственно выполняются процессором компьютера.  


5. **Flowchart — a diagram that shows a process, made up of boxes representing steps, decision, inputs and outputs.**
   - **True**  
   - **Перевод:** Блок-схема — диаграмма, отображающая процесс, состоящий из блоков, представляющих шаги, решения, входные и выходные данные.  


6. **IDE — it is visual tools that allow programmers to develop programs more efficiently. Commonly, an IDE may have a compiler, debugger, text editor, and other integrated tools.**
   - **True**  
   - **Перевод:** IDE (интегрированная среда разработки) — это визуальные инструменты, позволяющие программистам разрабатывать программы более эффективно. Обычно IDE включает компилятор, отладчик, текстовый редактор и другие интегрированные инструменты.  


7. **Interpreter — it is a collection of binary digits or bits that the computer reads and interprets.**
   - **False**  
   - Исправлено: Interpreter — a program that translates and executes source code line by line during runtime.  
   - **Перевод:** Интерпретатор — программа, которая переводит и выполняет исходный код строка за строкой во время выполнения.  


8. **String — a sequence of characters often stored as a variable in a computer program. These characters can include numbers, letters and symbols.**
   - **True**  
   - **Перевод:** Строка — последовательность символов, часто хранимая как переменная в компьютерной программе. Эти символы могут включать цифры, буквы и знаки.  


9. **Boolean — a data type that has one of two possible values (usually denoted true and false) is intended to represent the two truth-values of logic.**
   - **True**  
   - **Перевод:** Логический тип данных (boolean) — тип данных, который имеет одно из двух возможных значений (обычно true и false) и предназначен для представления двух логических значений истинности.  


10. **Mutable — not changing or unable to be changed.**
    - **False**  
    - Исправлено: Mutable — capable of being changed or altered after creation.  
    - **Перевод:** Изменяемый (mutable) — способный быть изменённым после создания.
